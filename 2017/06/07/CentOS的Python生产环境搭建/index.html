Python环境搭建：centos部署Python项目

技术
  • Python

安装Python环境

通过pyenv和virtualenv实现多个python版本并存

安装pyenv

$ git clone git://github.com/yyuu/pyenv.git ~/.pyenv
$ echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bashrc
$ echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bashrc
$ echo 'eval "$(pyenv init -)"' >> ~/.bashrc
$ exec $SHELL -l

查看可安装版本 

$ pyenv install --list

安装Python依赖包 

$ yum install readline readline-devel readline-static
$ yum install openssl openssl-devel openssl-static
$ yum install bzip2-devel bzip2-libs

安装Python3.6.0 

$ pyenv install 3.6.0 -v

更新数据库 

$ pyenv rehash

查看已安装版本

$ pyenv versions

更改全局Python版本 

$ pyenv global 3.6.0

安装virtualenv

创建目录

$ mkdir /var/devops/
$ cd /var/devops/

安装虚拟环境 

$ pip install virtualenv

创建项目需要的独立虚拟环境 

$ virtualenv -p /root/.pyenv/versions/3.6.0/bin/python --no-site-packages 
$ virtualenv-3.6.0

激活虚拟环境 

$ source /var/devops/virtualenv-3.6.0/bin/activate

退出虚拟环境

$ deactivate

设置用户登录后默认进入该虚拟环境 

在~/.bash_profile文件中添加
$ source /var/devops/virtualenv-3.6.0/bin/activate

让.bash_profile文件的修改生效
$ source ~/.bash_profile

安装参考：http://blog.csdn.net/chris__kk/article/details/45127973

拉取项目代码

生成SSH Key，配置部署秘钥

生成SSHkey 

$ ssh-keygen -t rsa -C "liuyating@yiguo.com"

一路回车，保存在默认路径/root/.ssh/下

登录GitLab，进入项目的设置选项，添加.ssh/id_rsa.pub的内容至gitlab项目的部署秘钥

拉取项目代码

创建项目仓库目录：

$ /var/devops/projects

拉取项目代码

$ cd /var/devops/projects/
$ git clone 项目的gitlab地址

安装依赖库：

$ pip install -r requirements.txt

安装mysqlclient报错

缺乏依赖，centos需安装mariadb-devel

$ yum install mariadb-devel -y

然后重新安装即可

项目配置运行

项目配置

上传项目配置文件，修改配置

$ cd /var/devops/projects/yiguo_devops/system/
$ rz命令上传settings文件
完善域名、数据库、API_URL等配置

进程管理

安装supervisor进程管理工具

由于supervisor不支持python3.4以上版本，但在python2.7环境下安装，用于python3也无异常。

先退出虚拟环境

$ deactivate

切换到系统默认的Python版本2.7 

$ pyenv global system
$ yum install python-setuptools
$ easy_install pip
$ pip install supervisor

安装成功后， 使用supervisor自带的命令生成模板

$ echo_supervisord_conf > supervisord.conf
$ cp supervisord.conf/etc/supervisord.conf

创建项目相关的配置文件

$ mkdir/etc/supervisord.d/
$ vim /etc/supervisord.conf
修改/etc/supervisord.conf配置
    [include]
    files = /etc/supervisord.d/*.conf

$ cd /etc/supervisord.d/
$ touch yiguo_devops.conf
配置gunicorn启动运行及log等

重新切回虚拟环境

启动supervisor

$ supervisord -c /etc/supervisord.conf

Nginx配置域名转发和负载均衡

安装nginx

$ yum install nginx

运行nginx

$ /usr/sbin/nginx

检测配置文件是否正常

$ /usr/sbin/nginx -t

编辑配置文件

$ vim /etc/nginx/conf.d/yiguo_devops.conf

修改配置后reload生效 

$ /usr/sbin/nginx -s reload
