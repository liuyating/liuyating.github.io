Python图像处理：centos安装Python图像处理库Pillow报错

技术
  • Python

安装依赖库

yum install zlib zlib-devel 
yum install libjpeg libjpeg-devel 
yum install freetype freetype-devel

安装Pillow

pip install Pillow
指定版本： pip install Pillow==3.4.2

如果下载较慢，可更换源，如豆瓣源。

pip install -i  https://pypi.douban.com/simple/ Pillow==3.4.2

依旧报错，例：

_imaging.c:3359: error: expected ?.?. ?.?. ?.?. ?.sm?.or ?._attribute__?.before ?.?.token

当需要安装一个源外的python类库, 而这个类库内含需要编译的调用python api的c/c++文件时还需要安装python-dev，在centos中应该是python-devel。

pip install python-devel

再安装Pillow，问题解决。
